* Install truffle and an ethereum client
  ```javascript
  npm install -g truffle
  npm install -g ethereumjs-testrpc
  ```

* Install node dependency.
  ```javascript
  npm install`
  ```
* Run testrpc
  ```sh
  testrpc
  ```

* Compile and migrate the contracts.
  ```javascript 
  truffle compile
  truffle migrate
  ```

* Run the webpack server.
  ```javascript
  npm run start
  ```
